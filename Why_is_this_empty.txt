This section is dedicated to present my enginners thesis project which was smart home automation model based on Raspberry Pi and Arduino Nano utilising free open source GUI and home automation system Domoticz. Whole concept was based on 3 Python scripts which were resposible for handling Alarm system, controlling lights, temeperature readings as well as controlling step motors. All scripts were triggered by Domoticz's GUI but they were made to run independently which was whole concept. Sadly due to University's policy I cannot share those scripts here.
